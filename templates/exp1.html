Experiment 1 : Switch Case Statement

Aim

To write a C++ program using the Switch Case statement to display eligibility for voting.

Objective

To understand the working of the switch-case decision-making statement in C++.

Theory

The switch statement is a multi-way selection statement. It compares the value of an expression with different case labels. When a matching case is found, that block is executed. The break statement terminates the switch block. If none of the cases match, the default block executes.

Algorithm

  1. Start the program.
  2. Declare an integer variable.
  3. Assign value 18.
  4. Use switch statement.
  5. Check each case.
  6. Display corresponding message.
  7. Terminate the program.

Program


#include <iostream>
using namespace std;

int main()
{
    int var = 18;

    switch(var)
    {

        case 15:

            cout << "You are a kid";
            break;

        case 18:

            cout << "Eligible for vote";
            break;

        default:

            cout << "Default Case is executed";
            break;

    }

    return 0;
}

Program Output


Eligible for vote

Result

Thus the C++ program using the Switch Case statement was executed successfully and the expected output was obtained.
