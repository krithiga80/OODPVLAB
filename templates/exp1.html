Experiment 1 : Switch Case Statement

Aim

To write a C++ program using the switch-case statement to determine whether a person is eligible to vote based on age.

Objective

To understand the working of the switch statement along with user input and decision making.

Theory

The switch statement selects one block of code from multiple alternatives. In this experiment, the user's age is first converted into a choice value (0 or 1), and the switch statement displays the corresponding message.

Algorithm

  1. Start the program.
  2. Declare an integer variable age.
  3. Read the age from the user.
  4. If age is greater than or equal to 18, assign choice = 1; otherwise assign choice = 0.
  5. Use a switch statement with choice.
  6. If choice is 1, display Eligible to Vote.
  7. If choice is 0, display Not Eligible to Vote.
  8. Stop the program.

Program

#include <iostream>
using namespace std;

int main()
{
    int age;

    cout << "Enter your age: ";
    cin >> age;

    int choice;

    if(age >= 18)
        choice = 1;
    else
        choice = 0;

    switch(choice)
    {
        case 0:
            cout << "Not Eligible to Vote";
            break;

        case 1:
            cout << "Eligible to Vote";
            break;

        default:
            cout << "Invalid Input";
    }

    return 0;
}
