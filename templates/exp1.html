Experiment 1: Switch Case Statement

Aim

To write a C++ program using the switch-case statement to display eligibility for voting.

Objective

To understand the working of the switch-case decision-making statement in C++.

Theory

The switch statement is a multi-way decision control statement. It allows a variable or expression to be tested against multiple constant values. When a matching case is found, the corresponding block of statements is executed. The break statement terminates the switch block. If none of the cases match, the default case is executed.

Algorithm

  1. Start the program.
  2. Declare an integer variable.
  3. Assign a value to the variable.
  4. Use the switch statement.
  5. Compare the variable with each case.
  6. Display the corresponding message.
  7. Terminate the program.
