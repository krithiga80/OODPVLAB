Experiment 2 : Pointers

Aim

To demonstrate the use of pointers by storing and accessing the address and value of a variable.

Objective

To understand pointer declaration, initialization and dereferencing in C++.

Theory

A pointer stores the memory address of another variable. The '&' operator obtains an address and the '*' operator accesses the value stored at that address.

Algorithm

  1. Start.
  2. Declare integer variable num.
  3. Read a number from the user.
  4. Declare pointer ptr.
  5. Store the address of num in ptr.
  6. Display the value of num, address of num, pointer value and value pointed to by the pointer.
  7. Stop.

Program

#include <iostream>
using namespace std;

int main()
{
    int num;
    cout << "Enter a number: ";
    cin >> num;

    int *ptr = &num;

    cout << "Value of num: " << num << endl;
    cout << "Address of num: " << &num << endl;
    cout << "Value of ptr: " << ptr << endl;
    cout << "Value pointed by ptr: " << *ptr << endl;

    return 0;
}
