Experiment 4 : Exception Handling

Aim

To demonstrate exception handling in C++ by checking division by zero.

Objective

To understand try, throw and catch blocks in C++.

Theory

Exception handling transfers control to a catch block whenever an error occurs. This improves program reliability.

Algorithm

  1. Start.
  2. Read two integers.
  3. Use a try block.
  4. If divisor is zero, throw a runtime exception.
  5. Otherwise perform division.
  6. Catch and display the exception message.
  7. Stop.

Program

#include <iostream>
#include <stdexcept>
using namespace std;
int main(){
 int x,y;
 cout<<"Enter dividend: "; cin>>x;
 cout<<"Enter divisor: "; cin>>y;
 try{
   if(y==0) throw runtime_error("Divide by zero exception");
   cout<<"Result = "<<x/y;
 }
 catch(const runtime_error &e){
   cout<<"Exception: "<<e.what();
 }
 return 0;
}
